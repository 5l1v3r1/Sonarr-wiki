# Linux

### Ubuntu/Debian
**Add NzbDrone's repository to your software source**
       
    sudo add-apt-repository 'deb http://update.nzbdrone.com/repos/apt/debian develop main'

**Install NzbDrone**
	
	sudo apt-get update
	sudo apt-get install nzbdrone 

**Start NzbDrone**

	mono /opt/NzbDrone/NzbDrone.exe
